# 日本橫濱之旅 2/27 - 3/3

Vue 3 + Vite + Tailwind CSS 單頁旅遊規劃網頁。

## 執行方式

```bash
npm install   # 若尚未安裝依賴
npm run dev   # 開發模式，瀏覽器開啟 http://localhost:5173
npm run build # 產出 dist/ 靜態檔，可部署至任意靜態主機
npm run preview # 預覽建置結果
```

## 功能摘要

- 響應式版面（手機、平板、桌面）
- 時間軸 + 卡片展示每日行程
- 各交通節點附「地圖」按鈕，一鍵開啟 Google Maps 搜尋
- 頁首「車票購買方式」說明 N'EX 成田特快來回票購買地點與官網連結
- 2/2 為預留卡片「自由探索中...」
